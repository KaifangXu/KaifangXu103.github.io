内置核心数据类型

Posted on 2018-02-24 | In Python Data Structures

python把数据视作解决问题过程的关键。
在python以及任何其他面向对象的编程语言中，我们定义一个类（class）来描述数据的状态（the state）以及描述这个数据能做什么（the behavior）.
Classes类似于抽象数据类型（abstract data types）因为一个class的用户只能看到一个数据（data item）的状态和behavior。

Data items 在面向对象的程序中被叫做objects
一个对象（object）是一个类（class）的例子

在这里我们省略介绍python中的内置numeric和boolean classes，主要说一下内置collection data types

Built-in collection data types

list

Operation Name Operator Explanation
indexing [ ] Access an element of a sequence
concatenation + Combine sequences together
repetition * Concatenate a repeated number of times
membership in Ask whether an item is in a sequence
length len Ask the number of items in the sequence
slicing [ : ] Extract a part of a sequence

Lists support a number of methods that will be used to build data structures. This Table provides a summary. Examples of their use follow.
以下表格是总结对于一个list可以进行的method操作
Method Name|Use|Explanation
:—–:|:—–:|:—–:
append|alist.append(item)|Adds a new item to the end of a list
insert|alist.insert(i,item)|Inserts an item at the ith position in a list
pop|alist.pop()|Removes and returns the last item in a list
pop|alist.pop(i)|Removes and returns the ith item in a list
sort|alist.sort()|Modifies a list to be sorted
reverse|alist.reverse()|Modifies a list to be in reverse order
del|del alist[i]|Deletes the item in the ith position
index|alist.index(item)|Returns the index of the first occurrence of item
count|alist.count(item)|Returns the number of occurrences of item
remove|alist.remove(item)|Removes the first occurrence of item
